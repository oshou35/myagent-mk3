# Retro office design

Date: 2026-07-20

## Direction

mk3 no longer follows the rich Gather-style dashboard direction. The target is a simple top-down retro RPG office like the attached references: tile-based ground, hard black outlines, saturated colors, small character sprites, simple command buttons, and an RPG message window.

## Visual rules

- Use a 32px grid and pixelated canvas rendering.
- Keep UI chrome minimal: top HUD, map, message window, command menu.
- Avoid right-side dashboards, floating cards, video-call UI, and search-heavy app chrome.
- Do not copy Kairosoft or stock image assets. Recreate only the structural style: tile map, office/town objects, character sprites, stat popups, and simple dialog.

## First-screen model

- Top HUD: day, money, trust, unresolved work, running count.
- Main field: Agent company office with facilities.
- Message window: selected agent/facility explanation.
- Command menu: HQ, jobs, agents, facilities, reports.

## Facilities

- Reception / job board: task intake and attention items.
- Work line: workflow execution.
- Manual shelf: procedures.
- QA room: verification gates.
- Library: knowledge and reports.
- Control room: queue runner.
- President room: safety approvals and external impact holds.
