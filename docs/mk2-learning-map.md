## Lessons to show as gameplay

- Missing metadata weakens reports, so mk3 needs a visible record-keeping mechanic.
- Codex handoff is useful when input/output contracts are clear, so mk3 should show delegation as a work-line assignment.
- Safety gates must stay visible even in a game UI.
- Knowledge gaps should appear as empty library shelves or improvement quests.
- Reports should feed back into facility upgrades and agent training.
